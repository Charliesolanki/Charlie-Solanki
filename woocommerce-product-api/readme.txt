A modern WordPress starter plugin which uses the [WP Emerge](https://github.com/htmlburger/wpemerge) framework.

== Description ==

This Plugin is developed by Manektech to create custom route for fetching the products from third party Website.
- In order to get the JSON Format of the products you need to access below URL:
{site_url}/wp-json/woocommerce-product-api/v1/products/
- Above URL will fetch first page from the product list and if you wish to access other page as well then you can add an extra paramter in above URL which is mention Below.
{site_url}/wp-json/woocommerce-product-api/v1/products/{page_number}
- Once Plugin is activated then it will add a new page named as "External Products" and a shortcode will get included in that page to get the data from json and show in that page.
